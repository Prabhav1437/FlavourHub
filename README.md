# FlavourHUB 

Premium vegetarian restaurant website built with HTML and CSS.

## Overview

A modern, fully responsive restaurant website featuring glassmorphism design, smooth animations, and an intuitive user experience.

## Pages

- **Home** - Hero section, weekly specials, statistics, testimonials
- **Menu** - 7 categories with sidebar navigation and active section highlighting
- **About** - Brand story, team profiles, and specialties
- **Contact** - Contact form, location details, and social links

## Tech Stack

- HTML5
- CSS3 (Grid, Flexbox, Variables, Backdrop Filters)
- Font Awesome 6.6.0
- Google Fonts (Inter, Bebas Neue)

## Features

- Glassmorphism UI with backdrop blur effects
- Responsive design (desktop, tablet, mobile)
- Interactive button states with pressed glass effects
- CSS-only menu scroll spy using :has() and :target
- Smooth transitions and animations
- Full-width mobile navigation
