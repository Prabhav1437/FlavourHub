# FlavourHUB 

**Premium Vegetarian Restaurant Website**

> *Where Modern Taste Meets Classic Soul*

FlavourHUB is a modern, fully responsive restaurant website showcasing premium vegetarian dining. Built with pure HTML and CSS, it features elegant design, smooth animations, and an immersive user experience.

---

## Features

### Design & UI
- **Modern Aesthetics** - Clean, contemporary design with glassmorphism effects
- **Elegant Color Palette** - Sophisticated green gradients (#9DC08B, #CAE0BC, #F8F5E9)
- **Fully Responsive** - Optimized for desktop, tablet, and mobile devices
- **Smooth Animations** - Subtle transitions and hover effects throughout
- **Custom Branding** - Professional logo with SVG/PNG assets and favicon
- **Premium Typography** - Inter font family for optimal readability

###  Pages

#### Home (`index.html`)
- **Hero Section** - Eye-catching dual-image layout with brand tagline
- **Weekly Delights** - Rotating special offers (Paneer Special, Dosa Delight, Biryani Bonanza, Weekend Feast)
- **Statistics Showcase** - 5+ years, 5K+ customers, 20+ dishes, 4.9★ rating
- **Experience Section** - Brand story with compelling imagery
- **Testimonials** - Customer reviews from food enthusiasts
- **Call-to-Action** - Reserve table and view menu buttons

####  Menu (`menu.html`)
- **6 Categories**: Starters, Main Course, Breads & Sides, Desserts, Drinks, Race Menu
- **Premium Items** - Detailed descriptions with pricing (₹160-₹340)
- **Race Jam Weekends** - Exclusive F1-themed menu (Friday-Sunday)
- **Interactive Cards** - Hover effects and elegant layouts
- **Signature Dishes** - Smoky Paneer Tikka, Veg Biryani, Dark Chocolate Fondant

#### About (`about.html`)
- **Brand Story** - Mission and values
- **Our Speciality** - 5 value propositions with visual cards
- **Team Profiles** - Meet the people behind FlavourHUB
- **Unified Design** - Gradient background with consistent styling

####  Contact (`contact.html`)
- **Contact Information** - Phone, email, address, and hours
- **Interactive Form** - JavaScript-validated contact form
- **Location Details** - Restaurant interior photo with address
- **Social Media** - Instagram, LinkedIn, GitHub links
- **Transport Info** - Parking, metro, and bus accessibility

###  Special Features
- **Weekly Specials** - Different themed meals for each day
- **Race Jam Weekends** - F1-inspired exclusive menu
- **Premium Drinks** - Exotic beverages (Saffron Iced Latte, Matcha Mint Fusion, Blue Butterfly Lemonade)
- **Gourmet Desserts** - Fusion items (Dark Chocolate Fondant, Rasmalai Cheesecake, Italian Gelato)
- **Hamburger Menu** - Mobile-friendly navigation
- **Smooth Scrolling** - Enhanced user experience

---

## Technology Stack

- **HTML5** - Semantic markup with accessibility features
- **CSS3** - Modern styling techniques
  - CSS Grid & Flexbox for layouts
  - CSS Variables for theming
  - Animations & Transitions
  - Media Queries for responsiveness
  - Backdrop filters for glassmorphism
- **Font Awesome 6.6.0** - Icon library
- **Google Fonts** - Inter font family
- **Unsplash** - High-quality food photography
